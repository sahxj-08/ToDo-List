# ToDo-List
A minimalist to-do list app designed to help users stay organized.

I built this project to practice clean UI logic and 
state management, while designing something for my daily use.

Feautures:
- Add and Delete Tasks
- Track Completed VS Uncompleted Tasks
- Clean, distraction-free UI
- Mobile-responsive layout
- Progressive Web App

TechStack:
- HTML/CSS, JavaScript
